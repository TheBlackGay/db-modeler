# DB Modeler 版本规划

## 当前版本 v0.1.1

### 已完成功能
- [x] 基础项目管理（创建、编辑、删除、导入导出）
- [x] 基础表管理（创建、编辑、删除、搜索）
- [x] 基础字段管理（创建、编辑、删除）
- [x] 主键和自增设置
- [x] SQL导出功能（支持MySQL和PostgreSQL）
