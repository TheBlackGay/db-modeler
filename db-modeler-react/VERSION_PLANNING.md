# DB-Modeler-React 版本规划

## 项目概述
DB-Modeler-React 是一个基于 React + Ant Design 的数据库模型设计工具，支持数据表的可视化设计、字段管理、索引配置等功能。项目采用 JSON 文件进行数据持久化，支持项目的导入导出功能。

## 技术栈
- React 18
- TypeScript
- Ant Design
- Redux Toolkit
- React Router
- Vite
